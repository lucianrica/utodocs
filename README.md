# Utodocs

# Development Commands

## build & run local 
```shell
    npm run dev
```


# Test Commands

## unit tests 
```shell
    npm run test --silent
```

## e2e tests 
```shell
    npm run e2e
```

## linting 
```shell
    npm run lint
```

# Production Build Commands

## Build Prod
```shell
    npm run build
```

## Run Prod
```shell
    npm run prod
```
